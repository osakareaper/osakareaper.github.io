# What we gonna build in this course?

By the end of this course, we gonna put together a multi-agency system that search for jobs, highlight skills in a resume and increases our odds to get an interview.

We gonna do that by leveraging four different agents:
- A Tech Job Researcher;
- A Personal Profiler for Engineers;
- A Resume Strategist for Engineers;
- An Engineering Interview Preparer;

For this, we gonna use a few tools, such:
- Search the internet;
- Read websites;
- Read resume;
- Perform RAG (Retrieval Augmented Generation) on resume;

**This gonna make sure we highlight the right skills for the job!**
